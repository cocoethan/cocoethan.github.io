"Graph Based Application of Search Algorithms Illustrated" in Python

Description:
This project was an individual project whose purpose was to illustrate graph-based searching algorithms, specifically Depth-First Search, Breadth-First Search, Greedy-Breadth-First Search, and A-Star Search. The searching algorithms act in a 50x50 grid based environment beginning at a start node whose position is defined randomly or specified, and searches for the final node that is also defined randomly or specified. The path must completely avoid 'obstacles' as defined in black on the grid. The path may also travel through 'turfs' as defined in green on the grid, but with a greater action cost. All project code is written in Python and uses the matplotlib python library to define the grid environment. Images of the program's output are included below.
